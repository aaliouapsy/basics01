HTML Common Tags
"!DOCTYPE html" tag
declares the document as HTML5
"html" tag
is the root element
"Head" tag
contains the mind / information about the page element: metadata — information about the page, not in the page.
"Body" tag
contains the visible page content
"Header" tag
h1 to h6 tags, h1 is the largest, h6 is the smallest
"P" tag
used to define a paragraph
"Div" tag
a container unit that encapsulates other page elements and divides the HTML document into sections
"Span" tag
used to group inline-elements in a document
"Table" tag
used to display data in a tabular form
"ul/ol/dl" tag
ul = unordered list / ol = ordered list / dl = description list
li = list item / dd = definition description
